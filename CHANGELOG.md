# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2026-03-06

### Added
- ⚛️ Next.js 16 frontend with TypeScript
- 🎨 Modern gradient UI with responsive design
- 🔌 REST API with Flask and Pydantic validation
- 🐳 Docker support for containerized deployment
- 📊 Real-time API health monitoring
- 🔒 CORS support for cross-origin requests
- 📝 Comprehensive documentation
- 🚀 Quick start scripts (start.bat, start.sh)
- 🧪 API testing suite

### Changed
- 📁 Reorganized project structure (frontend/backend separation)
- 🔧 Changed default port from 5000 to 8080 (Windows compatibility)
- 📦 Updated dependencies to latest versions
- 🎯 Improved feature ordering for model predictions

### Fixed
- 🐛 Feature name ordering issue in predictions
- 🔐 Port access restrictions on Windows
- 🌐 CORS configuration for frontend-backend communication

## [1.0.0] - 2026-03-04

### Added
- 🤖 Machine learning pipeline with 7 algorithms
- 📊 Advanced feature engineering
- 📓 Jupyter notebooks for exploration
- 🎯 Model training and evaluation scripts
- 📈 Performance metrics and visualization
- 💾 Model persistence with joblib
